# Get-SSHTrustedHost

## SYNOPSIS
List Host and Fingerprint pairs that Posh-SSH trusts.

## SYNTAX

```
Get-SSHTrustedHost
```

## DESCRIPTION
List Host and Fingerprint pairs that Posh-SSH trusts.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Get-SSHTrustedHost
SSHHost                                                     Fingerprint

-------                                                     -----------
```
