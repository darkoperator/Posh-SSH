# Get-PoshSSHModVersion

## SYNOPSIS
Gets the current installed version and the latest version of Posh-SSH.

## SYNTAX

```
Get-PoshSSHModVersion [<CommonParameters>]
```

## DESCRIPTION
Gets the current installed version and the latest version of Posh-SSH.

## EXAMPLES

### EXAMPLE 1
```
Get-PoshSSHModVersion

InstalledVersion                                  CurrentVersion
----------------                                  --------------
1.7                                               1.6
```
