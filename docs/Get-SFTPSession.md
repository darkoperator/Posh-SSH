# Get-SFTPSession

## SYNOPSIS
Get current SFTP Sessions that are available for interaction.

## SYNTAX

```
Get-SFTPSession [[-SessionId] <Int32[]>] [<CommonParameters>]
```

## DESCRIPTION
Get current SFTP Sessions that are available for interaction.

## EXAMPLES

### EXAMPLE 1
```
Get-SFTPSession

SessionId  Host                                            Connected
---------  ----                                            ---------
    0      192.168.1.180                                     True
```

## PARAMETERS
